# LETG SPIP

Squelette SPIP du site [letg.cnrs.fr](http://letg.cnrs.fr/), basé sur le kit graphique statique du CNRS, compatible avec SPIP 3.x et prenant en charge la publication d'un labo "mutli antennes".
